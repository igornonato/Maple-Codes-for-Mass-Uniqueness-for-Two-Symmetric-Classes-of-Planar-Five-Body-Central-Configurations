# Maple-Codes-for-Mass-Uniqueness-for-Two-Symmetric-Classes-of-Planar-Five-Body-Central-Configurations
This repository contains all the Maple codes developed for my PhD thesis. If you do not have a Maple license, you can also find PDF versions of the corresponding .mw files. The codes are organized into folders according to the section in which each Lemma, Proposition, or Theorem appears.
